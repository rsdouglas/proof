Wix and Squarespace make it easy to build a beautiful website. They’re less clear on how to handle testimonials — their native options are clunky, and most tutorials skip the part where you actually collect new testimonials from happy customers.

Here’s the complete guide: how to add testimonials to Wix or Squarespace, and how to make them actually do something useful for your business.

Option 1: Wix Built-in Testimonials Section

Wix has a “Testimonials” section you can add from the editor. Here’s how:

  1. In the Wix Editor, click Add (+) in the left panel
  2. Select Section → Testimonials
  3. Choose a layout and click Add to Page
  4. Click each testimonial card to edit the text, name, and photo

Pros: No third-party tools needed. Looks native.

Cons: You have to manually enter testimonials one by one. There’s no collection mechanism — no link to send customers, no approval workflow. Every time you get a new testimonial, you’re back in the editor.

Option 2: Squarespace Quote Block

Squarespace doesn’t have a dedicated testimonials section, but you can use the Quote block:

  1. Add a new block to any page
  2. Select Quote from the block menu
  3. Type the testimonial text
  4. Add a text block below for the customer’s name

For a full testimonials section, create a new page or use a Summary block with a Blog collection (more advanced, but allows filtering and layout control).

Pros: Clean typography. Native to Squarespace.

Cons: Same problem as Wix — fully manual. No collection, no workflow.

Option 3: Embed a SocialProof Widget (Recommended)

Both Wix and Squarespace support embedding custom code, which means you can use a dedicated testimonial tool like SocialProof. This gives you a collection link, an approval workflow, and a live widget that updates automatically.

Setup (10 minutes)

Step 1: Sign up at socialproof.dev — free, no credit card

Step 2: You’ll get a collection link (like socialproof.dev/collect/your-handle). Share this with customers via email, text, or your post-purchase thank-you page.

Step 3: When a customer submits a testimonial, you approve it in the SocialProof dashboard.

Step 4: Copy the widget embed code from your dashboard.

Embedding in Wix

  1. In the Wix Editor, click Add (+) → Embed → Custom Embeds → Embed a Widget
  2. Click Enter Code and paste your SocialProof embed snippet
  3. Resize and position the widget on your page
  4. Click Publish

Embedding in Squarespace

  1. In the page editor, add a new block
  2. Select Code (available on Business plan and above)
  3. Paste your SocialProof embed snippet
  4. Click Apply

The widget renders your approved testimonials with automatic updates — add a new testimonial in SocialProof, it appears on your site without touching the editor.

If you’re adding 2–3 testimonials once and never touching them again, the native option is fine.

If you want to continuously grow your social proof — sending collection links after purchases, adding new testimonials monthly — you want a dedicated tool.

Where to Place Testimonials on Your Wix or Squarespace Site

Placement matters as much as content. High-converting spots:

  • Homepage — below the hero, above the fold if possible
  • Pricing or services page — right before the “book now” or “buy” button
  • About page — testimonials build personal trust here
  • Contact page — reduce hesitation right before the inquiry

If you’re using a widget, you can embed it on multiple pages from the same SocialProof account.

Getting Your First Testimonials

The hardest part isn’t the tech — it’s getting customers to write something.

The simplest approach: send your SocialProof collection link to your 5 best customers with a short personal note. Something like:

“Hey [name] — I’m putting together some customer stories for my website. Would you mind sharing a quick note about your experience? Here’s a link: [your link]”

Most people who liked working with you will say yes if you ask directly. Automated prompts after purchase or service completion work well too.
