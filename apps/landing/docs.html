Documentation

Everything you need to add social proof to your site.

Quick start

1. Sign up at app.proof.app
2. Create a Collection Form and share the link with your customers
3. Approve submissions in your dashboard
4. Create a Widget and copy the embed code
5. Paste <script src="..."></script> anywhere on your site

Widget embed code

Add this one line where you want testimonials to appear:

<script src="https://api.proof.app/w/YOUR_WIDGET_ID"></script>

The widget auto-renders inside a <div id="proof-widget"> or creates one if not present.

Widget types

Grid — responsive card grid (default)
Wall — masonry-style wall of quotes
Carousel — horizontal scrolling carousel (Pro)
